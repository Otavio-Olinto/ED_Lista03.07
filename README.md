# ED_Lista03.07
## Somar Dígitos: Recursiva

Este desafio consiste em criar uma aplicação, em Java, que receba um número natural e retorne a soma de seus algarismos.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A soma deve ser realizada por uma função recursiva;
3. Declarar, via comentário, a condição de parada da função.
```
